# Daggerheart: Voidborne — русская версия

Русская пользовательская сборка неофициального модуля **Daggerheart: Voidborne** для Foundry VTT.

## Что переведено

- классы: Ассасин, Кровавый охотник, Драчун, Колдун, Ведьма и Призыватель;
- все 13 подклассов;
- классовые и подклассовые особенности;
- вопросы предыстории и связи персонажей;
- боевые стойки Драчуна;
- классовые предметы, подписи действий и связанные эффекты;
- названия и описания доменов Крови и Ужаса.

Противники, окружение и макросы оставлены на английском языке.

## Совместимость

- Foundry VTT: версии 13–14;
- игровая система: Daggerheart.

## Установка

1. Распакуйте папку `daggerheart-voidborne-ru` в каталог `Data/modules` вашего сервера Foundry VTT.
2. Перезапустите Foundry VTT.
3. Включите модуль **Daggerheart: Voidborne — Русская версия** в настройках мира.
4. Откройте компендиум **Voidborne: Классы и классовые опции**.

Русская сборка имеет отдельный идентификатор и может быть установлена рядом с оригинальным модулем. Чтобы не создавать дубликаты одних и тех же материалов, рекомендуется включать только одну из двух версий.
